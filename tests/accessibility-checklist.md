# Accessibility Test Checklist

* [ ] Color contrast verified via tool against actual background
* [ ] Minimum 44x44px touch targets implemented
* [ ] Atkinson Hyperlegible typeface configured properly
* [ ] Line length restricted to 65-75 characters
* [ ] Visible focus state logic implemented entirely distinct from hover
* [ ] Prefers-reduced-motion respected contextually
* [ ] No meaning conveyed entirely via animation or color alone
